# REAL_i — Building Real Intelligence

Welcome to the **REAL_i** codebase. REAL_i is an AI-first, system-oriented educational platform designed to build next-generation AI Engineers. Rather than teaching basic APIs and prompts, REAL_i focuses on engineering systems that learn, adapt, fail, improve, and handle real-world constraints.

---

## 🎯 Our Philosophy

> **"We Don't Teach AI Tools... We Build AI Engineers."**

In REAL_i, we believe that real intelligence is not a single prompt, a single model, or a single tool. **Real intelligence is a system.** A robust AI system integrates:
- **Data & Memory**: Dynamic context retrieval.
- **Reasoning & Planning**: Structured thought processes.
- **Decision Making & Feedback Loops**: Self-correction and learning.
- **Human Interaction & Safety**: Guardrails and clean collaboration interfaces.
- **Real-World Constraints**: Handling latency, cost, and API failures.

## 🛠️ Tech Stack & Key Features

- **Frontend Core**: React 19, Vite, TailwindCSS v4, React Router v7.
- **Interactions & Animations**: GSAP (GreenSock) for fluid transitions, Lucide React icons.
- **Rich Rendering**: KaTeX + Rehype/Remark for rendering complex mathematical formulas and LaTeX in markdown.
- **Design System**: Tailored dark-mode first design with premium glassmorphism aesthetics.
